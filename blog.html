Important FAQ's

What is the DOM?
The DOM (Document Object Model) is a way for JavaScript to interact with HTML pages. It represents the page as a tree of elements, so you can read, change, add, or remove elements using JS.
What is the difference between innerText and innerHTML?
innerText represents only the visible text inside an element, ignoring any HTML tags. innerHTML represents the content inside an element, including HTML tags, so it can create new elements dynamically.
What is the difference between inline events and JavaScript event listeners?
Inline events are written directly in the HTML element, which makes the code less organized. JavaScript event listeners are written in JS files and can attach multiple actions to the same element without cluttering HTML.
What is an event in JavaScript?
An event is something that happens on a webpage, like:
  1. Clicking a button (click)
  2. Typing in an input box (input or keyup)
  3. Moving the mouse (mouseover)
You can use JS to respond when these events happen.
